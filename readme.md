# nextJs

[官网](https://nextjs.org/docs/app/getting-started/installation )

| 文件名             | 作用                 |
| --------------- | ------------------ |
| `page.tsx`      | 页面组件（必须命名为 `page`） |
| `layout.tsx`    | 布局组件               |
| `loading.tsx`   | 页面加载时展示            |
| `error.tsx`     | 页面出错时展示            |
| `not-found.tsx` | 页面 404 时展示         |
| `template.tsx`  | 可重复渲染的 layout（少用）  |
